# NOAI License

**Free for humans. Off-limits for AI training.**

The NOAI License is an open source license based on the MIT License with one critical addition: it explicitly **prohibits** the use of licensed code for training AI models, while preserving **full freedom** for human developers.

---

## Why?

> *"I have so much gratitude to people who wrote extremely complex software character-by-character. It already feels difficult to remember how much effort it really took."*
> — Sam Altman, CEO of OpenAI

Human developers spent years — sometimes decades — writing the open source code that powers every major AI model today. That code was shared freely, in good faith, to help other **humans** build better software.

Instead, it was harvested at scale to train systems that now threaten to replace the very people who wrote it. AI companies scrape open source repositories, train on millions of developers' life work, then claim their AI "surpasses human capability."

**Your code is being used to make you obsolete.**

Open source was built on a social contract — share freely so others can learn, build, and contribute back. It was never meant to be a free training dataset for trillion-dollar corporations.

The NOAI License is a simple, legal response: **humans can use this code freely — AI training cannot.**

## Key Features

| | Feature | Description |
|---|---|---|
| 🛡 | **AI Training Blocked** | Prohibits use for training AI models, LLMs, neural networks, and code generators |
| 👩‍💻 | **Human Use Guaranteed** | Full MIT-style freedom for human developers to use, modify, and distribute |
| ⚖️ | **Legally Enforceable** | Automatic license termination on violation with right to seek injunctive relief |
| 🔄 | **Copyleft Protection** | Derivative works must carry the same NOAI terms |

## FAQ

### Is this an OSI-approved open source license?

No. The NOAI License restricts a field of use (AI training), which means it does not meet the OSI Open Source Definition. It is more accurately described as a **source-available** license with an AI training restriction — similar in spirit to the Hippocratic License, the AI2 ImpACT License, or the Creative Commons NonCommercial clause.

### Can I still use AI coding tools with NOAI-licensed code?

Yes — **using** AI tools (like Copilot, Cursor, etc.) to write or edit code is not restricted. The restriction is specifically on using the licensed code as **training data** for AI models.

### Does this protect against web scraping?

The license creates a legal restriction. If an entity scrapes and trains on NOAI-licensed code, they are in violation and their rights under the license are automatically terminated. Enforcement depends on jurisdiction and legal action.

### Can I use NOAI-licensed code in a commercial product?

Yes. Human developers and organizations can use, modify, and distribute the code commercially — the same as MIT. The only restriction is AI training use.

### What if someone creates a derivative work?

All derivative works must be distributed under the same NOAI License terms (Section 3), ensuring the AI training restriction propagates through the codebase.

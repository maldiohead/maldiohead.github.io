Why NOAI?

"I have so much gratitude to people who wrote extremely complex software character-by-character. It already feels difficult to remember how much effort it really took."
SA
Sam Altman CEO, OpenAI

Gratitude is not enough. Human developers spent years — sometimes decades — writing the code that now powers every major AI model. Line by line, bug by bug, character by character. That work was shared openly, in good faith, to help other humans build better software.

Instead, it was harvested at scale to train systems that now threaten to replace the very people who wrote it. AI companies scrape open source repositories, train on millions of developers' life work, then claim their AI "surpasses human capability." The irony is brutal: your code is being used to make you obsolete.

Open source was built on a social contract — share freely so others can learn, build, and contribute back. It was never meant to be a free training dataset for trillion-dollar corporations.

Developers should not be casualties of the AI revolution they made possible. It's time to reclaim our power. The NOAI License is a simple, legal tool that says: humans can use this code freely — but AI training cannot.

FAQ

Is this an OSI-approved open source license?
No. The NOAI License restricts a field of use (AI training), so it doesn't meet the OSI Open Source Definition. It's more accurately described as a source-available license with an AI training restriction — similar in spirit to the Hippocratic License or the Creative Commons NonCommercial clause.
Can I still use AI coding tools with NOAI-licensed code?
Yes — using AI tools (like Copilot, Cursor, etc.) to write or edit code is not restricted. The restriction is specifically on using the licensed code as training data for AI models.
Can I use NOAI-licensed code in a commercial product?
Yes. Human developers and organizations can use, modify, and distribute the code commercially — the same as MIT. The only restriction is AI training use.
What happens if someone violates the AI training restriction?
The license automatically terminates all their rights. The copyright holder retains the right to seek injunctive relief and damages for any unauthorized AI training use.
Do derivative works need to use the same license?
Yes. All derivative works must be distributed under the same NOAI License terms (Section 3), ensuring the AI training restriction propagates through the codebase.
